# 知识库更新模板

请阅读以下来源的分析结果，提取与"{topic}"相关的关键信息并整合到知识库。

**来源**：{source_name}
**分析目录**：{analysis_dir}
**知识库路径**：{kb_path}

**关注维度**：
{dimensions}

**提取要求**：
1. 直接相关的章节/文章及其核心观点（简要列出）
2. 关键理论家及其概念（与{topic}相关的）
3. 重要文献线索（值得追踪的参考文献）
4. 可引用的核心论述（2-3句话的精炼表述）
5. 可用的理论框架

**整合到知识库时**：
- 新的理论框架：添加到「一、理论框架与核心概念」
- 新的关键概念：添加到概念术语表
- 可引用段落：添加到「四、可引用段落」
- 核心文献：添加到「三、核心文献追踪」
- 更新日志：在「五、更新日志」添加条目

**注意**：
- 只提取与{topic}相关的内容，不要泛泛摘抄
- 每条信息标注来源
- 累积性更新，不覆盖旧内容
- 检查是否有与已有内容重复或矛盾的地方
